# Power-BI-Dashboard-Project-
Owner of Store wants us to help them create a dashboard to track and analyze their online sales across India.
Complete Power BI project using retail store sales data.
I am pleased to share my first Power BI project, “Sales Data Analysis” The objective of this project is to deliver actionable insights to the store management team, enabling data-driven decisions to enhance sales performance.

This project involves collecting and analyzing sales data from multiple sources, performing data cleaning using Power Query Editor, building a robust data model, creating DAX measures, and developing an interactive, visually compelling dashboard. The dashboard highlights key performance indicators (KPIs) such as Total Amount, Total Profit, Total Quantity, and Average Order Value.

The scope of the project includes analyzing sales data over a defined period (e.g., quarterly) and comparing performance with previous periods to identify trends and patterns. It also focuses on evaluating sales by product category, identifying top-selling products, and analyzing total profit by sub-category.

The expected outcomes include uncovering opportunities to improve sales performance, optimizing product assortment and pricing strategies, and enhancing customer experience to drive loyalty and retention. By equipping the management team with meaningful insights, this project aims to increase revenue, improve operational efficiency, and support overall business growth.
